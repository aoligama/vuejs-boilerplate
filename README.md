# 🚀 Boilerplate Vue.js + Vuetify

This is a boilerplate with Vue.js + Vuetify

## 🛠 Tools
- Vue.js
- Vuetify
- Axios
- Vue Router
-  Vuex



## ⚡️ How to Use

Clone this repo or import the code on your own repo.

## ⚙️ Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Lints and fixes files
```
npm run lint
```
